üìù Word Add-in Installation Guide

Different methods to install your Word add-in, depending on your Word version.

Method 1: Insert ‚Üí Get Add-ins (Most Common)

  1. Open Microsoft Word
  2. Click the Insert tab in the ribbon
  3. Look for Get Add-ins (or Office Add-ins)
  4. Click My Add-ins at the top
  5. Look for Upload My Add-in (usually at the bottom)
  6. Browse to: C:\Users\msokrin\Document project\manifest.xml
  7. Click Upload

Method 2: Insert ‚Üí Add-ins (Alternative)

  1. Open Microsoft Word
  2. Click the Insert tab
  3. Click Add-ins (instead of Get Add-ins)
  4. Click My Add-ins
  5. Look for Upload My Add-in or Sideload Add-in
  6. Select your manifest.xml file

Method 3: Developer Tab (If Available)

First, enable Developer tab:

  1. File ‚Üí Options ‚Üí Customize Ribbon
  2. Check Developer in the right column
  3. Click OK

Then install add-in:

  1. Click Developer tab
  2. Click Add-ins
  3. Look for upload or sideload option

üîß Alternative: Manual Registry Method

If upload options are not available, try this:

  1. Close Word completely
  2. Open Command Prompt as Administrator
  3. Run: reg add HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\WEF\Developer /v AllowWebRequestsFromThisOrigin /t REG_SZ /d "*"
  4. Run: reg add HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\WEF\Developer /v AllowHttpsRequestsFromThisOrigin /t REG_SZ /d "*"
  5. Restart Word
  6. Try the upload methods again

Method 4: Office 365 Admin (If Available)

For Office 365 Business users:

  1. Go to admin.microsoft.com
  2. Settings ‚Üí Integrated apps
  3. Upload custom apps
  4. Upload your manifest.xml

üåê Web Version Alternative

If desktop Word doesn't work, try Word Online:

  1. Go to office.com
  2. Open Word Online
  3. Insert ‚Üí Add-ins ‚Üí My Add-ins ‚Üí Upload My Add-in
  4. Upload your manifest.xml

‚úÖ Success Indicators

Once installed, you should see:

  • A new "Hello Group" section in the Home ribbon
  • A "Hello Moti" button in that section
  • Clicking it opens a task pane on the right
  • The add-in will appear in ALL Word documents

Quick Test

Before installing, verify these work:

  • Task Pane Test - Should load the add-in interface
  • Manifest Test - Should download the manifest file
  • API Test - Should return {"status":"ok"}

üîÑ Still Having Issues?

  3. Try a different Word document
  4. Check if your organization allows add-ins
  5. Try Word in Safe Mode (hold Ctrl while starting)
